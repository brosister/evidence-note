# 증거노트 개인정보처리방침 초안

최종 수정일: 2026-04-03

증거노트(이하 "앱")는 개인 약속, 거래, 정산 기록을 사용자의 기기 안에서 정리하고 관리할 수 있도록 돕는 앱입니다.

## 1. 수집하는 정보
앱은 다음 정보를 사용자가 직접 입력하거나 첨부할 수 있도록 제공합니다.
- 제목, 상대 이름, 금액, 날짜, 메모
- 사진 첨부 파일
- 음성 녹음 파일
- 손서명 이미지
- 연락처 연동 시 사용자가 선택한 연락처 정보

## 2. 정보의 이용 목적
수집 또는 저장된 정보는 다음 목적에 사용됩니다.
- 약속/거래/정산 기록 생성 및 관리
- 타임라인 및 증거 요약 표시
- 미해결 상태 관리 및 후속 알림 제공
- PDF 생성 및 공유 기능 제공
- 사용자가 직접 선택한 연락처 연결

## 3. 저장 방식
현재 앱 초안 기준으로 기록 데이터와 첨부 파일은 사용자의 기기 내 로컬 저장소에 보관됩니다.
서버 전송 기능이 도입되는 경우, 관련 내용은 정책에 별도로 업데이트됩니다.

## 4. 권한 사용 안내
앱은 기능 제공을 위해 아래 권한을 요청할 수 있습니다.
- 연락처 권한: 상대방을 빠르게 선택하고 연결하기 위해 사용
- 사진 접근 권한: 캡처 이미지 및 증거 사진 첨부를 위해 사용
- 마이크 권한: 음성 증거 녹음을 위해 사용
- 알림 권한: 약속 확인 및 미회수 확인 알림 제공을 위해 사용

## 5. 제3자 제공
앱은 사용자의 개인정보를 외부에 판매하거나 임의로 제3자에게 제공하지 않습니다.
단, 사용자가 직접 공유 기능을 사용해 PDF나 텍스트, 이미지를 외부 앱으로 전송하는 경우 해당 전송은 사용자의 명시적 행동에 따라 이루어집니다.

## 6. 사용자 통제
사용자는 앱 내에서 기록을 수정하거나 삭제할 수 있습니다.
로컬 저장 구조상, 사용자가 삭제한 데이터는 기기 내 저장소에서 제거될 수 있습니다.

## 7. 아동의 개인정보
앱은 일반 사용자를 대상으로 하며, 아동을 주요 대상으로 설계되지 않았습니다.

## 8. 법적 효력 관련 안내
앱은 개인 기록 정리 및 공유를 돕는 도구입니다.
앱 내 기록, 타임스탬프, 해시, 첨부 정보는 참고용 정리 기능이며 특정 법적 효력이나 공증 수준을 보장하지 않습니다.
